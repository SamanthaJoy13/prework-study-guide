HTML

  • Hypertext Markup Language-builds the webpage frame.
  • Element: entire element from opening to closing tab.
  • Tag: only what appears inside the angle brackets.
  • The head element contains information about the webpage.
  • Inside the head element is the title, style, base, link, & meta elements.
  • The body element represents the visable content shown to the user.
  • Child elements are elements that are nested inside other elements.
  • Parent elements are the element that holds all the children.
  • Child elements inside the body include P, ul, ol, li, img, br, & a.
  • Semantic HTML elements are the elements that describes its content and the role it plays on a webpage.
  • Not all elements are semantic (ex: div & span).

CSS

  • Cascading Style Sheets-defines how webpage content is presented to users.
  • Inline CSS: apply to a specific HTML tag.
  • Internal style sheet: CSS rules for the webpage in the head section of the HTML file.
  • External style sheet: allows developers to keep all their CSS rules in a seperate file.
  • The selector defines the element or attribute to which the rules, or declarations will apply.
  • Declarations contain the CSS property we want to apply and the value of the property.
  • The margin property is the space around the outside of an element.
  • The padding property is the space around the content inside an element.
  • * applies a rule to all the elements visable on the page.
  • 2 or more elements that will share the same styling can be grouped together.

Git

  • Git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Var is a keyword that declares a variable.
  • console.log method outputs a message to the web console by adding an argument.
  • Control flow is the order in which a computer executes code in a script.
  • If statement is a way of introducing decision-making into the code(will only execute if true).
  • Array: a single variable that is used to hold a group of data.
  • Index: number that identifies the location of an item in the array.
  • For loop performs a task on all items in an array by allowing a single code block to be executed over & over.
  • Commenting out code makes the code inactive without deleting it.
  • A function is a set of instructions that tells the computer how to perform a certain task.
